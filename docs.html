The Blockwerk documentation

The blockwerk uses Lua 5.1, see Lua Official Website

When using the API, you'll notice that there are a few changes—you can use bit32, Instance, Vector2, Vector3, and even CoordinateFrame!


Lua Additional functions
  • task.spawn (threadOrCoroutine)

    Creates a new thread to run before the scheduler finishes executing the last thread.
    [Note] This function can cause an infinite number of threads to be created, which may prevent the scheduler from ever finishing executing a thread.

  • task.defer (threadOrCoroutine)

    Creates a new thread that will only run once the scheduler has finished executing all existing threads

  • task.wait (seconds)

    Puts the thread to sleep; if you don't provide any parameters, your thread will simply be paused and will only run after the scheduler has finished executing all the threads

  • table.freeze (table)

    Freezes the table provided as an argument to the function. Note: You cannot unfreeze the table after freezing it, unless you create a new table.

  • table.isfrozen (table)

    Checks whether the table is frozen
